What to match

Senders / domains, one per line i One entry per line:
• spam@example.com — exact sender address
• *@newsletter.com — a whole domain (wildcard)
• annoying.com — a whole domain (bare form)
• mail.annoying.com — only that subdomain
• lines starting with # are comments
one entry per line · reuse across runs

Options

“Save senders CSV…” exports the last listing — you choose where to save it. With dry-run on, “Run” only previews.
